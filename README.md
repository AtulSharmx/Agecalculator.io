# Age Calculator

A simple web-based Age Calculator made using HTML, CSS, and JavaScript.  
This project takes the user's date of birth and calculates their exact age in years, months, and days.

## Live Demo

https://agecalculator-io.vercel.app/

---

## Features

- Calculate exact age instantly
- Shows age in years, months, and days
- Simple and clean user interface
- Responsive design
- Fast and easy to use

---

## Technologies Used

- HTML
- CSS
- JavaScript
